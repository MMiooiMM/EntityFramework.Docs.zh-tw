# <a name="saving-data"></a>儲存資料

每個內容執行個體的 `ChangeTracker` 都負責追蹤寫入至資料庫所需的變更。 當您變更實體類別的執行個體時，這些變更會記錄在 `ChangeTracker` 中，然後在您呼叫 `SaveChanges` 時寫入至資料庫。 資料庫提供者負責將變更轉譯成資料庫特定作業 (例如關聯式資料庫的 `INSERT`、`UPDATE` 和 `DELETE` 命令)。
