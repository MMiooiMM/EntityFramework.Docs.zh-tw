# <a name="ef-core-tools--extensions"></a>EF Core 工具和延伸模組

工具和延伸模組提供 Entity Framework Core 的額外功能。

延伸模組是透過各種來源所建置。 並非所有延伸模組都會維護為 Entity Framework Core 專案的一部分。 考慮使用協力廠商延伸模組時，請務必評估品質、授權、支援等，確保它們符合您的需求。
