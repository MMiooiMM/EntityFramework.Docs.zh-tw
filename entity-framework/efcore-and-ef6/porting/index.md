# <a name="porting-from-ef6-to-ef-core"></a>從 EF6 移植至 EF Core

因為 EF Core 在本質上有所變更，我們不建議嘗試將 EF6 應用程式移至 EF Core，除非您有充足的理由進行該變更。 您應該將從 EF6 移至 EF Core 視為移植，而非升級。
